Summary

Find configuration macros used in #ifdex/#ifndef/#elif lines.

Searches a C or C++ code tree attempting to report a list of configuration macros - that is, symbols set by autoconf or some other configuration-management and/or build system during compiles. Also useful for catching stray and broken guard symbols.
